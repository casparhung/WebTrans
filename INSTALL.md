# WebTrans 安裝指南

WebTrans 是一款網頁翻譯擴充功能。本指南說明如何在 Chrome 或 Microsoft Edge 安裝。

---

## 安裝步驟

### Chrome

1. **下載並解壓**
   - 下載 `webtrans-release-1.0.1.zip`
   - 右鍵 → 解壓全部
   - 記住解壓後資料夾位置（例如 `C:\Users\YourName\Downloads\webtrans`）

2. **開啟 Chrome 擴充管理頁面**
   - 在網址列輸入：`chrome://extensions/`
   - 按 Enter

3. **啟用開發人員模式**
   - 右上角找到「開發人員模式」開關
   - 點選打開（會變藍色）

4. **載入未封裝項目**
   - 點左上角「載入未封裝項目」
   - 選擇剛才解壓的資料夾
   - 點「選擇資料夾」

5. **完成！**
   - WebTrans 會出現在擴充清單
   - 建議點列表下方的釘選圖示，讓 ICON 固定在工具列

---

### Microsoft Edge

1. **下載並解壓**
   - 下載 `webtrans-release-1.0.1.zip`
   - 右鍵 → 解壓全部
   - 記住解壓後資料夾位置

2. **開啟 Edge 擴充管理頁面**
   - 在網址列輸入：`edge://extensions/`
   - 按 Enter

3. **啟用開發人員模式**
   - 左下角找到「開發人員模式」開關
   - 點選打開

4. **載入未封裝項目**
   - 點左上角「載入未封裝項目」
   - 選擇剛才解壓的資料夾
   - 點「選擇資料夾」

5. **完成！**
   - WebTrans 會出現在擴充清單
   - 建議點右邊的釘選圖示，讓 ICON 固定在工具列

## 使用方式

### 基本操作

1. **整頁翻譯**
   - 按 WebTrans 工具列圖示開啟 popup
   - 選擇目標語言
   - 按「翻譯整頁」

2. **選取文字翻譯**
   - 滑鼠選取網頁上的文字
   - 會自動出現浮動翻譯提示

3. **恢復原文**
   - 按「還原原文」或點頁面右下角浮動按鈕

4. **拖曳懸浮按鈕**
   - 按住頁面右下角的「譯」/「原」按鈕不放
   - 拖到想要的位置
   - 位置會自動記憶

### AI 翻譯（可選）

1. **啟用 AI 模式**
   - 開啟 WebTrans popup
   - 勾選「AI 模型翻譯」
   - 選擇預設或填入自己的設定

2. **快速填入**
   - **Ollama**：本機執行，無需 API Key
   - **OpenAI**：需要 API Key（付費）
   - **Groq**：需要 API Key（部分免費）

3. **自訂設定**
   - API Base URL：例如 `https://api.openai.com/v1`
   - API Key：例如 `sk-...`（Ollama 可留空）
   - Model：例如 `gpt-4o-mini`、`llama2`

4. **測試連線**
   - 填完後按「測試 AI 連線」
   - 若顯示綠色「連線成功」，表示設定正確
   - 若失敗，會自動改用 Google 翻譯

## 注意事項

1. **無帳號系統**
   - WebTrans 不需要登入或建立帳號
   - 所有設定存在瀏覽器本機

2. **隱私保護**
   - 翻譯內容會送往翻譯服務（Google 或你指定的 AI API）
   - 不會儲存翻譯歷史

3. **更新擴充**
   - 若要更新版本，刪除舊版再載入新版資料夾
   - 或直接覆蓋資料夾後在擴充頁面按更新

4. **遇到問題**
   - 試試重新載入擴充（擴充清單上按更新圖示）
   - 若特定網站不工作，可能是該網站有強制樣式干擾

## 常見問題

**Q: 我該如何移除擴充？**  
A: 開啟 `chrome://extensions/`（或 `edge://extensions/`），找到 WebTrans，點右下角的「移除」。

**Q: 拆機式安裝（載入未封裝）有限制嗎？**  
A: 有。每次重啟瀏覽器時，警告列會提示「未確認的擴充」，但不影響使用。若想移除提示，需上架到官方商店。

**Q: 可以在多台電腦上安裝嗎？**  
A: 可以，每台電腦都獨立執行，設定不同步。

**Q: 使用 Google 翻譯需要付費嗎？**  
A: 不需要，免費使用。但大量翻譯可能被限流。若要穩定服務，可考慮改用 AI 翻譯或官方 Google Translate API。
